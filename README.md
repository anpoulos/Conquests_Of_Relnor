# Conquests_Of_Relnor
A GameMaker project.

Controls:

-General
t - talk to/activate object
shift - use current wewapon
space - run

-Commands
r (hold) + left click (hold) - selection box
r (hold) + right click (hold) - deselection box
left click - select
right click - deselect

-Selection Commands
f - square form selected
e - enable engage
ctrl + e - disable engage
q - deselect all
